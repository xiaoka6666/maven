Maven Repository

Maven access point for custom baritone builds.

Usage

Add this repository to your build.gradle:

repositories {
    maven {
        name = "xiaoka6666Maven"
        url = "https://xiaoka6666.github.io/maven/"
        artifactUrls("https://xiaoka6666.github.io/maven/cabaletta/baritone-unoptimized/${project.name}")
    }
}
